Description

Nifty is a responsive admin dashboard template based on Bootstrap v5.1.x framework. It comes with a lot of valuable components. Build with SCSS (SCSS files are included). Easily maintain CSS with variables, mixins, and more.

Rebuilt from scratch in 2022

We created this theme in 2014 and rebuilt it from scratch in 2022 to support the latest version of Bootstrap v5.1. Use the latest modern techniques and CSS properties such as grid, flex, inset, etc.

Features

  • Fully responsive theme ready to customize and publish
  • SASS files included for easy customization
  • Mobile-friendly layout built with a mobile-first design.
  • Build in innovative navigation options such as Expanded, Collapsed, and Offcanvas based on your settings.
  • Super Flexible layout based on your choice of settings.
  • Includes 72 color schemes and unlimited possibilities when set in SCSS.
  • 2000+ high-quality Premium Icons split into two styles: Solid and Line.
  • 40+ Premium Background Images.
  • Pre-built page layouts such as Email, File Manager, Taskboard, etc.
  • 8 transition effects for a smooth and unique transition.

Images

  • The Premium Background is a high-quality image and is included in the package with permission from the author.
  • Avatar images are premium assets created by our partners and used exclusively on Nifty Admin Theme.
  • All the images used in the demo are brought to you by Unsplash.
